# 小结

实际上就是在开始编译和编译结束的时候，通过 Date.now() 获取对应的时间戳，放在了 Stats 对象上（其实还是通过 compilation 拿到的）

Compiler.js

// ...
const startTime = Date.now() // 获取开始编译时间
// ...
compilation.startTime = startTime;
compilation.endTime = Date.now() // 获取编译结束时间
// ...
